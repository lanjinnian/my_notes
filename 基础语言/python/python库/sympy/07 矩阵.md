# 矩阵的引入与创建

在使用矩阵之前，需要从 `sympy` 导入 `Matrix` 类以及相关的特殊矩阵函数。SymPy 中的矩阵默认是可变的（Mutable），如果需要不可变矩阵，可以使用 `ImmutableMatrix`。
```python
from sympy import Matrix, eye, zeros, ones, diag, symbols
```

## 基础创建

最常见的创建方式是通过传入一个“列表的列表”（list of lists）。
```python
# 创建一个 2x2 矩阵
M = Matrix([[1, 2], [3, 4]])
# 创建一个列向量 (等价于 3x1 矩阵)
V = Matrix([1, 2, 3]) 
```

对应的数学表示为：
$M = \begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix}, \quad V = \begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix}$

## 特殊矩阵构造器

SymPy 提供了快速构造常见矩阵的方法：
- **单位矩阵：** `eye(n)` 创建一个 $n \times n$ 的单位矩阵。
- **零矩阵：** `zeros(r, c)` 创建一个 $r$ 行 $c$ 列的全零矩阵。
- **全 1 矩阵：** `ones(r, c)` 创建一个 $r$ 行 $c$ 列的全 1 矩阵。
- **对角矩阵：** `diag(1, 2, 3)` 创建对角线元素为 1, 2, 3 的矩阵。`diag` 也可以用来拼接块矩阵（Block Matrices）。

# 基本属性与操作

## 形状与索引
与 NumPy 类似，SymPy 矩阵支持基于 0 的索引。
```python
M.shape  # 返回元组 (rows, cols)，例如 (2, 2)
M[0, 1]  # 获取第 1 行、第 2 列的元素 (值为 2)
```

## 行与列的提取
```python
M.row(0)  # 提取第一行，返回 Matrix([[1, 2]])
M.col(-1) # 提取最后一列，返回 Matrix([[2], [4]])
```

## 基础数学运算

矩阵支持标准的算术运算符（要求维度匹配）：
- **加减法：** `M + N`, `M - N`
- **矩阵乘法：** `M * N` (注意：这是矩阵乘法，不是逐元素乘法)
- **标量乘法：** `3 * M`
- **矩阵求幂：** `M**2` (等价于 `M * M`)
- **求逆：** `M**-1` (等价于 `M.inv()`)

##  转置
```python
M_T = M.T
```

### 三、 核心线性代数运算 (Linear Algebra)

这是 SymPy 矩阵模块最核心的价值所在，支持符号计算。

#### 1. 行列式 (Determinant)

计算矩阵的行列式：

Python

```
M = Matrix([[1, 2], [3, 4]])
det_M = M.det()  # 返回 -2
```

#### 2. 行最简形 (Reduced Row Echelon Form - RREF)

`rref()` 方法返回一个元组：`(简化后的矩阵, 主元列的索引元组)`。

Python

```
M = Matrix([[1, 2, 3], [2, 4, 6], [1, 1, 1]])
M_rref, pivots = M.rref()
# M_rref 为 Matrix([[1, 0, -1], [0, 1, 2], [0, 0, 0]])
# pivots 为 (0, 1)
```

#### 3. 零空间与列空间 (Nullspace & Columnspace)

- **零空间：** `M.nullspace()` 返回一个列表，包含构成零空间基的列向量。
    
- **列空间：** `M.columnspace()` 返回一个列表，包含构成列空间基的列向量。
    

#### 4. 特征值与特征向量 (Eigenvalues & Eigenvectors)

- **特征值：** `M.eigenvals()`
    
    - 返回一个字典：`{特征值: 代数重数}`。
        
    - 例如 `{3: 1, -2: 1}` 表示特征值 3 和 -2 的代数重数均为 1。
        
- **特征向量：** `M.eigenvects()`
    
    - 返回一个列表，列表元素为元组：`(特征值, 代数重数, [特征向量列表])`。
        

#### 5. 对角化 (Diagonalization)

如果矩阵 $M$ 可对角化，`diagonalize()` 返回矩阵 $P$ 和对角矩阵 $D$，使得 $M = P D P^{-1}$。

Python

```
P, D = M.diagonalize()
# 验证： P * D * P**-1 == M
```

### 四、 矩阵微积分 (Matrix Calculus)

SymPy 允许矩阵元素为符号变量，从而支持微积分操作。

#### 1. 符号矩阵

Python

```
x, y = symbols('x y')
F = Matrix([x**2 + y, 2*x*y])
```

#### 2. 雅可比矩阵 (Jacobian)

如果要计算向量函数对变量的雅可比矩阵，可以使用 `jacobian()` 方法：

Python

```
X = Matrix([x, y])
J = F.jacobian(X)
```

上述代码将计算：

$J = \begin{bmatrix} \frac{\partial (x^2+y)}{\partial x} & \frac{\partial (x^2+y)}{\partial y} \\ \frac{\partial (2xy)}{\partial x} & \frac{\partial (2xy)}{\partial y} \end{bmatrix} = \begin{bmatrix} 2x & 1 \\ 2y & 2x \end{bmatrix}$

### 五、 求解线性方程组

给定方程组 $Ax = b$，在 SymPy 中可以使用 `LUsolve` 或 `solve` 等方法直接求解：

Python

```
A = Matrix([[1, 2], [3, 4]])
b = Matrix([5, 6])

# 方法 1：使用矩阵逆 (不推荐用于大型或病态系统)
x = A.inv() * b

# 方法 2：使用 LUsolve (推荐)
x = A.LUsolve(b)
```
