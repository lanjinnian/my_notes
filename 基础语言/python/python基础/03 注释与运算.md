## 1. Python3 注释 (Comments)

注释是写给程序员看的，Python 解释器在执行代码时会完全忽略它们。合理使用注释可以极大地提高代码的可读性。

### 单行注释

使用井号 `#`。可以将注释放在代码的上方，也可以放在代码的正后方。

Python

```
# 这是一个单行注释，解释接下来的代码用途
print("Hello")  # 这也是注释，通常用于解释当前行的逻辑
```

### 多行注释

使用三个单引号 `'''` 或三个双引号 `"""`。虽然它们本质上是未赋值的字符串常量，但在实际开发中常被用作多行注释，特别是在编写函数或类的说明文档（Docstring）时。

Python

```
"""
这是一个多行注释的示例。
你可以在这里写很长的说明，
比如算法的思路、参数的定义等。
"""
```

## 2. Python3 运算符 (Operators)

Python 支持极其丰富的运算符，以下按照功能进行了分类梳理。

### ① 算术运算符

用于执行基本的数学运算。

|**运算符**|**描述**|**实例**|
|---|---|---|
|`+`|加|`10 + 20` 输出 `30`|
|`-`|减|`10 - 20` 输出 `-10`|
|`*`|乘|`10 * 20` 输出 `200`|
|`/`|除 (**总是返回浮点数**)|`10 / 2` 输出 `5.0`|
|`%`|取模 (返回除法的余数)|`10 % 3` 输出 `1`|
|`**`|幂运算|`2 ** 3` 输出 `8`|
|`//`|取整除 (向下取整)|`10 // 3` 输出 `3`|

### ② 比较 (关系) 运算符

用于比较两个值，返回的结果一定是布尔值 (`True` 或 `False`)。

|**运算符**|**描述**|**实例 (假设 a=10, b=20)**|
|---|---|---|
|`==`|等于 (比较**值**是否相等)|`a == b` 返回 `False`|
|`!=`|不等于|`a != b` 返回 `True`|
|`>`|大于|`a > b` 返回 `False`|
|`<`|小于|`a < b` 返回 `True`|
|`>=`|大于等于|`a >= 10` 返回 `True`|
|`<=`|小于等于|`b <= 10` 返回 `False`|

### ③ 赋值运算符

用于给变量赋值，最基本的是等号 `=`。

- **复合赋值**：将算术运算与赋值结合。例如 `a += 2` 等价于 `a = a + 2`。同理还有 `-=`, `*=`, `/=`, `%=`, `**=`, `//=`.
    
- **海象运算符 `:=`** (Python 3.8+ 新增)：允许在表达式内部为变量赋值。
    
    Python
    
    ```
    # 传统写法
    n = len(a)
    if n > 10:
        print(f"太长了，共有 {n} 个元素")
    
    # 使用海象运算符 (让代码更紧凑)
    if (n := len(a)) > 10:
        print(f"太长了，共有 {n} 个元素")
    ```
    

### ④ 逻辑运算符

用于组合多个条件判断语句。

|**运算符**|**逻辑**|**描述**|
|---|---|---|
|`and`|与|只有两个条件**都为** `True` 时，结果才为 `True`。（具有**短路**特性：如果第一个条件为 False，直接返回 False，不再计算第二个条件）|
|`or`|或|只要有一个条件为 `True`，结果就为 `True`。（短路特性：如果第一个条件为 True，直接返回 True）|
|`not`|非|逻辑反转。如果条件为 True，返回 `False`。|

### ⑤ 成员运算符

Python 独有的优雅语法，用于测试一个元素是否包含在序列（如字符串、列表、元组或字典的键）中。

- **`in`**: 如果在指定的序列中找到值，返回 `True`。
    
- **`not in`**: 如果没有在指定的序列中找到值，返回 `True`。
    
    Python
    
    ```
    my_list = [1, 2, 3]
    print(2 in my_list)      # 输出: True
    print(5 not in my_list)  # 输出: True
    ```
    

### ⑥ 身份运算符

用于比较两个对象的**内存地址**（即判断它们是不是同一个对象）。

- **`is`**: 判断两个标识符是不是引用自同一个对象。
    
- **`is not`**: 判断两个标识符是不是引用自不同的对象。
    

> **核心易错点：`is` 和 `==` 的区别！**
> 
> - `==` 比较的是两个变量的**值**是否相等。
>     
> - `is` 比较的是两个变量的**内存地址** (id) 是否相同。
>     
> 
> Python
> 
> ```
> a = [1, 2, 3]
> b = a[:] # 切片会复制出一个新列表
> 
> print(a == b) # True (它们里面的值一模一样)
> print(a is b) # False (它们在内存中是两个不同的列表)
> ```
