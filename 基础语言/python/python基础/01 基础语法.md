### 1. 编码与标识符

**默认编码：**

Python 3 源码文件默认使用 **UTF-8** 编码，这意味着你可以直接在代码中使用中文作为变量名（虽然在实际开发中极其不推荐这样做）。

**标识符（命名规则）：**

- 由字母、数字、下划线 `_` 组成。
    
- **不能以数字开头**（如 `1var` 是错误的，`var1` 是正确的）。
    
- **严格区分大小写**（`Runoob` 和 `runoob` 是两个不同的变量）。
    
- 不能使用 Python 的保留字（关键字）。
    

**查看保留字：**

Python

```
import keyword
print(keyword.kwlist)
# 输出如: ['False', 'None', 'True', 'and', 'as', 'assert', 'break', 'class', 'continue', 'def', ...]
```

### 2. 注释与代码块（缩进）

这是 Python 区别于 C、Java 等语言最显著的特征。

**注释：**

- **单行注释**：使用井号 `#`。
    
- **多行注释**：使用三个单引号 `'''` 或三个双引号 `"""`。
    

Python

```
# 这是一个单行注释
print("Hello, World!") 

'''
这是多行注释
可以写很多行
'''
```

**缩进 (Indentation)：**

Python 不使用大括号 `{}` 来控制类、函数以及其他逻辑判断，而是**严格使用缩进**来写代码块。

- 缩进的空白数量是可变的，但**同一个代码块的语句必须包含相同的缩进空格数**（通常约定俗成为 **4 个空格**）。
    
- 如果缩进不一致，程序会直接报错 `IndentationError`。
    

Python

```
if True:
    print("True")  # 缩进 4 个空格
else:
    print("False") # 缩进 4 个空格
```

### 3. 多行语句

通常 Python 的一行就是一条语句。但如果语句很长，可以使用反斜杠 `\` 来实现多行语句。

Python

```
total = item_one + \
        item_two + \
        item_three
```

**例外情况**：在 `[]`, `{}`, 或 `()` 中的多行语句，不需要使用反斜杠 `\`。

Python

```
my_list = ['item_one', 'item_two', 
           'item_three', 'item_four']
```

### 4. 数据类型与字符串基础

Python 3 中有 6 个标准的数据类型（Number 数字、String 字符串、List 列表、Tuple 元组、Set 集合、Dictionary 字典）。这里重点提一下最常用的字符串语法：

- 单引号 `'` 和双引号 `"` 用法完全相同。
    
- 三引号 `'''` 或 `"""` 可以指定一个多行字符串。
    
- **转义符 `\`**：如 `\n` 代表换行。
    
- **自然字符串 (Raw String)**：在字符串前面加上 `r` 或 `R`，表示按原样输出，不发生转义。
    

Python

```
word = '字符串'
sentence = "这是一个句子。"
paragraph = """这是一个段落，
可以跨越多行。"""

print(r"这里的 \n 不会换行，会原样输出")
```

### 5. 基础输入与输出

**输出 `print()`：**

`print` 默认输出是换行的。如果想要实现不换行，需要在末尾加上 `end=""` 参数。

Python

```
print("Hello", end=" ")
print("World")
# 输出结果: Hello World
```

**输入 `input()`：**

执行到 `input()` 时，程序会暂停，等待用户在控制台输入文本，按下回车键后继续。**接收到的所有输入都会被当作字符串 (String) 处理**。

Python

```
user_name = input("请输入你的名字: ")
print("你好, " + user_name)
```

### 6. 导入模块 (Import)

在 Python 中，你需要导入其他文件或内置模块来使用它们的功能。

- **`import somemodule`**: 将整个模块导入。
    
- **`from somemodule import somefunction`**: 从某个模块中导入特定的函数或变量。
    
- **`from somemodule import *`**: 将某个模块中的所有函数导入（通常不推荐，容易造成命名空间污染）。
    

Python

```
# 导入整个 sys 模块
import sys
print(sys.path)

# 只导入 sys 模块中的 argv 变量
from sys import argv
print(argv)
```
