### 1. 六大标准数据类型概览

Python3 中有 6 个标准的数据类型，它们按照**是否可变（Mutable）**被划分为两大类：

- **不可变数据 (Immutable)**：一旦创建，内存中的值不能被修改。
    
    - **Number (数字)**
        
    - **String (字符串)**
        
    - **Tuple (元组)**
        
- **可变数据 (Mutable)**：内存中的值可以被修改。
    
    - **List (列表)**
        
    - **Dictionary (字典)**
        
    - **Set (集合)**
        

> **技巧**：你可以使用内建函数 `type(var)` 来查看变量的类型，或者使用 `isinstance(var, type)` 来判断变量是否属于某个特定的类型。

### 2. 基础数据类型详解

#### ① Number (数字)

Python3 支持四种不同的数值类型：

- **`int` (整型)**：Python3 中只有长整型，没有 Python2 中的 `Long`。可以表示任意大小的整数。
    
- **`float` (浮点型)**：带有小数部分的数字。
    
- **`bool` (布尔型)**：`True` 和 `False`。在 Python3 中，布尔值其实是整型的子类，`True` 相当于 `1`，`False` 相当于 `0`，它们甚至可以进行数学运算。
    
- **`complex` (复数)**：如 `1 + 2j`，由实数部分和虚数部分构成。
    

#### ② String (字符串)

字符串由单引号 `'` 或双引号 `"` 括起来。

- **截取 (切片)**：语法为 `变量[头下标:尾下标:步长]`。正向索引从 `0` 开始，反向索引从 `-1` 开始。
    
- **运算**：加号 `+` 用于字符串拼接，星号 `*` 用于字符串重复。
    

Python

```
word = 'Python'
print(word[0:2])  # 输出: Py (截取索引0到2，不包含2)
print(word[-1])   # 输出: n (获取最后一个字符)
print(word * 2)   # 输出: PythonPython
```

#### ③ List (列表)

列表是 Python 中使用最频繁的数据类型。它写在方括号 `[]` 之间，元素可以是不同的数据类型（甚至可以嵌套列表）。

- 列表完全支持像字符串那样的切片操作。
    
- 列表是**可变的**，可以通过索引直接修改元素。
    

Python

```
my_list = [1, 'hello', 3.14, True]
my_list[0] = 100  # 修改第一个元素
print(my_list)    # 输出: [100, 'hello', 3.14, True]
```

#### ④ Tuple (元组)

元组写在小括号 `()` 里，元素用逗号隔开。它和列表非常相似，但**元组的元素不能修改**。

- 主要用于保护数据不被意外更改，或者作为字典的键。
    
- **注意**：构造包含 1 个元素的元组时，必须加逗号，否则会被当作普通的括号运算符。例如 `tup = (50,)`。
    

#### ⑤ Set (集合)

集合是由一个或数个形态各异的大小整体组成的，构成集合的事物或对象称作元素。它使用大括号 `{}` 或 `set()` 函数创建。

- **特点**：集合中的元素是**无序**且**不重复**的（自带去重功能）。
    
- **注意**：创建一个空集合必须用 `set()` 而不是 `{}`，因为 `{}` 是用来创建空字典的。
    
- 支持集合的数学运算，如差集 `-`，并集 `|`，交集 `&`，对称差集 `^`。
    

Python

```
a = set('abracadabra') # 自动去重
print(a) # 输出可能为: {'a', 'r', 'b', 'c', 'd'} (每次顺序可能不同)
```

#### ⑥ Dictionary (字典)

字典是无序的 **键(key) : 值(value)** 集合。写在大括号 `{}` 里。

- **键 (key)**：必须是唯一的，且必须是**不可变类型**（如字符串、数字或元组）。
    
- **值 (value)**：可以是任意数据类型。
    
- 字典是通过键来存取数据的，而不是通过偏移量（索引）。
    

Python

```
my_dict = {'name': 'Alice', 'age': 25}
print(my_dict['name']) # 输出: Alice
```

### 3. 数据类型转换

在实际编程中，我们经常需要将数据从一种类型转换为另一种类型。类型转换分为**隐式类型转换**和**显式类型转换**。

#### 隐式类型转换 (自动转换)

在进行数学运算时，Python 会自动将低精度类型转换为高精度类型以避免数据丢失。

Python

```
num_int = 123
num_float = 1.23
# 整数和浮点数相加，Python会自动将整数123转为浮点数123.0
result = num_int + num_float 
print(type(result)) # 输出: <class 'float'>
```

#### 显式类型转换 (强制转换)

这是我们在写代码时最常用的方式，通过调用内置的转换函数来实现：

- **`int(x)`**: 将 x 转换为一个整数。如果是浮点数，会**直接丢弃**小数部分。如果是纯数字字符串可以转换，但包含字母或小数点的字符串（如 `"12.3"`）直接用 `int()` 会报错。
    
- **`float(x)`**: 将 x 转换为一个浮点数。
    
- **`str(x)`**: 将对象 x 转换为字符串表示。
    
- **`list(x)`**: 将序列 x（如字符串、元组、集合）转换为一个列表。
    
- **`tuple(x)`**: 将序列 x 转换为一个元组。
    
- **`set(x)`**: 将序列 x 转换为一个集合（同时达到去重的效果）。
    
- **`dict(d)`**: 创建一个字典，d 必须是一个包含 `(key, value)` 元组的序列。
    

Python

```
# 字符串转整数
s = "100"
n = int(s)

# 列表去重的小技巧 (列表 -> 集合 -> 列表)
raw_list = [1, 2, 2, 3, 3, 3]
unique_list = list(set(raw_list)) 
print(unique_list) # 输出: [1, 2, 3]
```
