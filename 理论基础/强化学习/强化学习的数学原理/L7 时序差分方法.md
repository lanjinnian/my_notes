## 1. 时序差分（TD）学习简介

时序差分（TD）学习是强化学习中最知名且核心的方法之一 。它是继蒙特卡洛（MC）之后的第二种**免模型（Model-free）**方法 。

- **数学本质**：TD算法可以被视为求解给定策略 $\pi$ 的**贝尔曼方程**（Bellman Equation）的随机逼近（SA）算法 。
    
- **核心优势**：
    
    - **在线更新（Online）**：获得奖励后可立即更新，无需等待回合结束 。
        
    - **适用连续任务**：由于其在线特性，TD可处理非终止性的持续任务 。
        
    - **低方差**：相比MC，TD涉及的随机变量较少，因此估计方差更低 。

## 2. 状态价值的 TD 学习

### 2.1 算法描述

更新公式为：

$$v_{t+1}(s_t) = v_t(s_t) - \alpha_t(s_t) [v_t(s_t) - (r_{t+1} + \gamma v_t(s_{t+1}))]$$

- **TD 目标 (TD Target)**：$\overline{v}_t = r_{t+1} + \gamma v(s_{t+1})$。算法将当前估计驱动向该目标 。
    
- **TD 误差 (TD Error)**：$\delta_t = v(s_t) - \overline{v}_t$。它反映了当前估计值与真实价值之间的偏差，代表了从新经验中获得的“创新”信息 。
    

### 2.2 收敛性

在满足步长条件（$\sum \alpha = \infty, \sum \alpha^2 < \infty$）且每个状态被访问足够多次的前提下，TD算法以概率 1 收敛到 $v_\pi(s)$ 。

## 3. 动作价值的 TD 学习：Sarsa 及其变体

### 3.1 Sarsa (On-policy)

Sarsa 是 TD 学习的动作价值版本，旨在求解动作价值形式的贝尔曼方程 。

- **公式**：$q_{t+1}(s_t, a_t) = q_t(s_t, a_t) - \alpha_t [q_t(s_t, a_t) - (r_{t+1} + \gamma q_t(s_{t+1}, a_{t+1}))]$ 。
    
- **命名由来**：算法每一步涉及 $(s_t, a_t, r_{t+1}, s_{t+1}, a_{t+1})$ 序列 。
    
- **策略改进**：将 Sarsa 估值与 $\epsilon$-greedy 策略结合，可实现策略搜索并逼近最优策略 。
    

### 3.2 Expected Sarsa

Expected Sarsa 将 TD 目标中的下个状态动作价值替换为**期望值**：

$$\text{Target} = r_{t+1} + \gamma \sum_a \pi_t(a|s_{t+1}) q_t(s_{t+1}, a)$$

- **优点**：通过对动作求期望减少了随机性，方差比 Sarsa 更低 。
    

### 3.3 n-step Sarsa

n-step Sarsa 统一了 Sarsa（1步）和蒙特卡洛（$\infty$步）方法 。

- **特点**：使用 n 步奖励和第 n 个状态的估值作为目标。n 较大时性能接近 MC（高方差低偏差），n 较小时接近 Sarsa（低方差高偏差）。

## 4. 最优动作价值学习：Q-learning (Off-policy)

Q-learning 是应用最广的强化学习算法之一，直接求解**贝尔曼最优方程** 。

- **核心公式**：
    
    $$q_{t+1}(s_t, a_t) = q_t(s_t, a_t) - \alpha_t [q_t(s_t, a_t) - (r_{t+1} + \gamma \max_{a} q_t(s_{t+1}, a))]$$
    
- **TD 目标**：$r_{t+1} + \gamma \max_{a} q_t(s_{t+1}, a)$。它与 Sarsa 的唯一区别在于此处取最大值而非实际执行的动作价值 。
    

---

## 5. 同策（On-policy）与 异策（Off-policy）

这是 TD 学习中非常重要且易混淆的概念 ：

- **行为策略 (Behavior Policy)**：用于与环境交互、产生样本的策略 。
    
- **目标策略 (Target Policy)**：不断更新并期望最终达到最优的策略 。
    
- **判别**：
    
    - **On-policy (如 Sarsa, MC)**：行为策略与目标策略相同 。
        
    - **Off-policy (如 Q-learning)**：两者不同。它允许算法利用任何策略产生的样本来寻找最优策略，且行为策略可以保持探索性 。

## 6. 统一视角总结

所有上述算法均可表示为统一形式：$q_{t+1} = q_t - \alpha(q_t - \overline{q}_t)$，差异仅在于 **TD 目标 $\overline{q}_t$** 的定义不同 ：

|**算法**|**TD 目标 q​t​ 的表达式**|
|---|---|
|**Sarsa**|$r_{t+1} + \gamma q_t(s_{t+1}, a_{t+1})$|
|**Expected Sarsa**|$r_{t+1} + \gamma \sum_a \pi_t(a|
|**Q-learning**|$r_{t+1} + \gamma \max_a q_t(s_{t+1}, a)$|
|**n-step Sarsa**|$r_{t+1} + \gamma r_{t+2} + \dots + \gamma^n q_t(s_{t+n}, a_{t+n})$|
|**Monte Carlo**|$r_{t+1} + \gamma r_{t+2} + \gamma^2 r_{t+3} + \dots$|

这些算法都在以随机逼近的方式求解对应的贝尔曼方程（BE）或贝尔曼最优方程（BOE） 。
