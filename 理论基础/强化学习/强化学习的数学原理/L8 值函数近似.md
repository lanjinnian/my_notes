## 1. 课程动机：从表格表示到函数表示

在之前的课程（如蒙特卡洛、TD学习）中，状态/动作价值均采用**表格（Tabular）**形式进行存储和更新。

- **表格方法的局限性**：
    
    - **存储瓶颈**：当状态空间或动作空间极大，甚至是连续空间时，表格无法存储所有价值（维度灾难）。
        
    - **泛化能力差（Lack of Generalization）**：表格中每个状态的价值是独立更新的。未被访问过的状态，其价值无法得到更新。
        
- **函数近似方法（Function Approximation）**：
    
    - **核心思想**：引入一个参数化的函数 $\hat{v}(s, w)$ 或 $\hat{q}(s, a, w)$ 来近似真实的价值函数，其中 $w$ 是参数向量（例如神经网络的权重）。
        
    - **优势**：极大地节省了存储空间；具备**泛化能力**，一旦参数 $w$ 通过部分访问过的状态得到更新，未访问过状态的价值也会自动更新。
        

---

## 2. 状态价值近似的目标函数 (Objective Function)

为了找到最优的参数 $w$ 使得 $\hat{v}(s, w)$ 最接近真实的策略状态价值 $v_\pi(s)$，我们需要定义一个目标函数。

### 2.1 均方价值误差 (Mean Squared Value Error, MSVE)

目标函数定义为估计值与真实值差值的均方期望：

$$J(w) = \frac{1}{2} \mathbb{E}_{S \sim d_\pi} \left[ (v_\pi(S) - \hat{v}(S, w))^2 \right]$$

- **随机变量 $S$ 的分布**：期望是对状态 $S$ 求的，其概率分布为 $d_\pi(s)$。这是策略 $\pi$ 下马尔可夫链的**平稳分布（Stationary Distribution）**。
    
- **含义**：它表示在长期运行中，智能体处于各个状态的概率。这意味着我们更关心那些经常被访问的状态的近似精度。
    

---

## 3. 状态价值的参数更新 (State Value Approximation)

优化的目标是寻找 $w$ 最小化 $J(w)$。这可以结合前两讲学到的**随机梯度下降（SGD）**来实现。

### 3.1 理想的随机梯度下降

计算 $J(w)$ 对 $w$ 的梯度：

$$\nabla_w J(w) = -\mathbb{E} \left[ (v_\pi(S) - \hat{v}(S, w)) \nabla_w \hat{v}(S, w) \right]$$

采用 SGD 更新参数 $w$：

$$w_{t+1} = w_t + \alpha \left[ v_\pi(s_t) - \hat{v}(s_t, w_t) \right] \nabla_w \hat{v}(s_t, w_t)$$

- **痛点**：在实际中，我们根本**不知道真实的 $v_\pi(s_t)$**，因此上述理想公式无法直接使用。
    

### 3.2 结合 TD 学习的近似更新

为了解决不知道真实价值的问题，我们使用 **TD 目标（TD Target）** 来替代 $v_\pi(s_t)$：

- **TD 目标**：$\bar{v}_t = r_{t+1} + \gamma \hat{v}(s_{t+1}, w_t)$
    
    将 TD 目标代入更新公式，得到**带函数近似的 TD 算法**：
    
    $$w_{t+1} = w_t + \alpha \big[ r_{t+1} + \gamma \hat{v}(s_{t+1}, w_t) - \hat{v}(s_t, w_t) \big] \nabla_w \hat{v}(s_t, w_t)$$
    
- **TD 误差 $\delta_t$**：中间的括号项即为 TD 误差。
    

---

## 4. 动作价值的函数近似 (Action Value Approximation)

类似于状态价值，我们可以用 $\hat{q}(s, a, w)$ 近似动作价值 $q_\pi(s, a)$，从而进行策略改进并寻找最优策略。

### 4.1 Sarsa 与价值函数近似 (On-policy)

将函数近似应用于 Sarsa，更新公式变为：

$$w_{t+1} = w_t + \alpha \big[ r_{t+1} + \gamma \hat{q}(s_{t+1}, a_{t+1}, w_t) - \hat{q}(s_t, a_t, w_t) \big] \nabla_w \hat{q}(s_t, a_t, w_t)$$

- **数据需求**：需要完整的 $(s_t, a_t, r_{t+1}, s_{t+1}, a_{t+1})$ 经验序列。
    

### 4.2 Q-learning 与价值函数近似 (Off-policy)

将函数近似应用于 Q-learning，利用贝尔曼最优方程的思想，TD 目标取动作空间的最大值：

$$w_{t+1} = w_t + \alpha \big[ r_{t+1} + \gamma \max_{a'} \hat{q}(s_{t+1}, a', w_t) - \hat{q}(s_t, a_t, w_t) \big] \nabla_w \hat{q}(s_t, a_t, w_t)$$

- **数据需求**：只需 $(s_t, a_t, r_{t+1}, s_{t+1})$ 即可更新。

## 5. 深度强化学习的基石：DQN (Deep Q-Network)

当用**深度神经网络（DNN）**作为函数近似器时，直接使用上述 Q-learning 更新会导致训练极不稳定甚至发散。经典算法 **DQN** 引入了两个关键技术来解决这一问题：

1. **经验回放 (Experience Replay)**：
    
    - 将智能体收集到的经验样本 $(s, a, r, s')$ 存入经验池（Replay Buffer）。
        
    - 训练时从中**随机小批量（Mini-batch）采样**进行更新。这打破了序列样本间的高时间相关性，使数据分布更接近独立同分布（i.i.d）。
        
2. **目标网络 (Target Network)**：
    
    - 在计算 TD 目标 $r + \gamma \max \hat{q}(s', a', w^-)$ 时，使用一个独立的旧参数网络 $w^-$（目标网络）。
        
    - $w^-$ 定期从正在训练的主网络 $w$ 同步。这避免了“追逐移动靶”的问题，大幅提升了学习的稳定性。

## 6. 总结 (Summary)

- **方法跨越**：价值函数近似使得强化学习从离散、小规模的问题，跨越到了能够处理高维状态空间（如自动驾驶图像输入、围棋棋盘）的现代深度强化学习（Deep RL）领域。

**数学连贯性**：

- **SGD (第6讲)** 提供了参数更新的优化框架。
    
- **TD 学习 (第7讲)** 提供了逼近真实价值的数据目标。
    
- **函数近似 (第8讲)** 将两者结合，形成了能在复杂环境中实际部署的核心算法（如 DQN）。
