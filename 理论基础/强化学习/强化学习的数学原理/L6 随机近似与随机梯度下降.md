## 1. 均值估计的迭代实现 (Mean Estimation)

均值估计是强化学习的基础，因为状态或动作价值通常被定义为期望值 。

- **计算挑战**：传统的非增量方法需要收集所有样本后统一计算平均值，这导致在样本采集期间无法获得估计结果 。
    
- **迭代公式**：通过代数推导，均值估计可以转化为增量形式 ：
    
    $$w_{k+1}=w_{k}-\frac{1}{k}(w_{k}-x_{k})$$
    
- **算法优势**：
    
    - 每获得一个新样本 $x_k$，即可立即更新均值估计 $w_{k+1}$ 。
        
    - 虽然初始估计不准确，但随着样本增加，估计值会逐渐趋向于真实期望 $\mathbb{E}[X]$ 。
        
- **通用形式**：将 $1/k$ 替换为通用的学习率系数 $\alpha_k > 0$ 。

## 2. Robbins-Monro (RM) 算法

RM 算法是随机逼近（SA）领域的先驱性工作，旨在通过有噪声的数据解决寻根或优化问题 。

### 2.1 问题定义

- **目标**：寻找方程 $g(w)=0$ 的根 $w^*$ 。
    
- **核心困难**：函数 $g(w)$ 的表达式通常是未知的（如神经网络），只能获得观测值 $\tilde{g}(w_k,\eta_k)=g(w_k)+\eta_k$，其中 $\eta_k$ 为噪声 。
    

### 2.2 算法公式与哲学

- **算法更新**：$w_{k+1}=w_{k}-a_{k}\tilde{g}(w_{k},\eta_{k})$ 。
    
- **哲学思想**：在没有模型（即函数表达式）的情况下，依靠数据进行迭代 。
    

### 2.3 收敛条件 (RM 定理)

若满足以下三个条件，算法将以概率 1 收敛到 $w^*$ ：

1. **梯度限制**：$0 < c_1 \le \nabla_{w}g(w) \le c_2$，保证 $g(w)$ 单调递增且根唯一，同时梯度有上界 。
    
2. **步长约束**：
    
    - $\sum_{k=1}^{\infty}a_{k}=\infty$：确保步长不会过快收敛到 0，使算法有能力到达任意远的根 。
        
    - $\sum_{k=1}^{\infty}a_{k}^{2}<\infty$：确保步长最终趋于 0 以抵消随机噪声的影响 。
        
3. **噪声性质**：噪声 $\eta_k$ 的条件期望为 0，且方差有限 。

## 3. 随机梯度下降 (Stochastic Gradient Descent, SGD)

SGD 是 RM 算法在优化问题中的特殊应用 。

### 3.1 优化目标

- **目标函数**：最小化 $J(w)=\mathbb{E}[f(w,X)]$ 。
    
- **等效根寻找**：该优化问题可转化为寻找 $\nabla_{w}J(w)=\mathbb{E}[\nabla_{w}f(w,X)]=0$ 的根 。
    

### 3.2 三种梯度下降方法对比

|**方法**|**更新公式**|**核心差异**|
|---|---|---|
|**GD (梯度下降)**|$w_{k+1}=w_{k}-\alpha_{k}\mathbb{E}[\nabla_{w}f(w_{k},X)]$|使用真实梯度的期望值 。|
|**BGD (批量GD)**|$w_{k+1}=w_{k}-\alpha_{k}\frac{1}{n}\sum_{i=1}^{n}\nabla_{w}f(w_{k},x_{i})$|每次迭代使用数据集中所有样本 。|
|**SGD (随机GD)**|$w_{k+1}=w_{k}-\alpha_{k}\nabla_{w}f(w_{k},x_{k})$|每次迭代仅随机抽取一个样本，是 $n=1$ 的特殊情况 。|
|**MBGD (小批量GD)**|$w_{k+1}=w_{k}-\alpha_{k}\frac{1}{m}\sum_{j\in\mathcal{I}_{k}}\nabla_{w}f(w_{k},x_{j})$|每次迭代随机抽取 $m$ 个样本 。|

### 3.3 SGD 的收敛模式

- **收敛性质**：当 $w_k$ 距离最优解 $w^*$ 较远时，SGD 的表现非常接近 GD，能够快速接近目标区域 。
    
- **随机波动**：当 $w_k$ 接近 $w^*$ 时，由于单个样本梯度的随机性，收敛过程会表现出更多的随机震荡 。

## 4. 总结与关联 (Summary)

- **数学纽带**：均值估计可以看作是一个特殊的 SGD 过程，而 SGD 本质上是一个特殊的 RM 算法过程 。
    
- **核心价值**：这些算法不依赖于模型的具体表达式，而是基于数据的反馈进行迭代更新 。
    
- **后续影响**：随机逼近的思想为下一讲“时序差分（TD）学习”提供了理论框架，因为 TD 算法在表达形式上与 SA 算法高度相似 。
