
# 符号运算

- 符号计算处理的是以符号形式进行的数学对象的计算
- 这意味着数学对象被精确地表示，而不是近似地表示，并且带有未求值变量的数学表达式保持符号形式
- 符号结果可以进行符号简化, 能够一符号的方式进行各种运算
- 表达式不会计算出结果，而是输出符号化的结果
- 除了明显的简化外, 大多数简化不会自动执行
- sympy包含了用于包含了用于绘图、打印（如数学公式的二维漂亮打印输出，或 ）、代码生成、物理、统计、组合学、数论、几何、逻辑等的模块

> sympy的优点: 开源免费\轻量级\作为库使用


# sympy注意事项

## 符号

- 在 SymPy 中，变量不会自动定义。要定义变量，我们必须使用 `symbols`
- `symbols` 接受一个由空格或逗号分隔的变量名组成的字符串，并从中创建符号; 然后，我们可以将这些符号赋值给变量名
- 符号的名称和它所赋值的变量名称之间不需要有任何关系, 符号可以有超过一个字符的名字
```python
>>> a, b = symbols('b a')
>>> a
b
>>> b
a
```

- 如果一个变量被改变，已经用该变量创建的表达式不会自动改变
> [!NOTE]
> ```python
> x = symbols('x')
   expr = x + 1
   print(expr)
> ```
> 其结果为x + 1 , 原因在于:
> 将 `x` 改为 `2` 对 `expr` 没有影响。这是因为 `x = 2` 将 Python 变量 `x` 改为 `2`，但对我们在创建 `expr` 时使用的 SymPy 符号 `x` 没有影响。当我们创建 `expr` 时，Python 变量 `x` 是一个符号。创建后，我们将 Python 变量 `x` 改为 2。但 `expr` 保持不变。

- 在表达式中更改符号的值，使用 `subs`

## 等号

- 在 SymPy 中，`==` 表示精确的结构相等性测试, 只有在结构上完全相等是才会返回`Ture`
- 如果$a = b$，那么$a - b = 0$ 。因此，检查是否相等的最佳方法是取$a - b$并简化它，看看它是否变为 0。

## 其它事项

- 使用 `**` 进行幂运算, `^` 表示逻辑异或
- 每当你结合一个SymPy对象和一个SymPy对象，或者一个SymPy对象和一个Python对象时，你会得到一个SymPy对象
- 每当你结合两个Python对象时，SymPy永远不会介入，因此你会得到一个Python对象
