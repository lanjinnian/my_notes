# 关于等式表达的底层逻辑

- **等式的声明**：在 SymPy 中，符号等式不能使用 Python 原生的 `=` 或 `==`，必须使用 **`Eq(左边, 右边)`**。例如：`Eq(x, y)`。
- **“默认等于零”机制**：求解器默认将所有传入的非 `Eq` 表达式视为等于 0。
- 在使用等式求解的时候建议使用`solveset(x**2 - 1, x)`

# 求解代数方程

## 单变量求解：`solveset`
```python
solveset(equation, variable=None, domain=S.Complexes)
```
(默认在复数域内求解)

> [!NOTE] 
> **返回类型**：
>    - **有确切解**：返回集合 `FiniteSet`、`Interval`（区间）或带通项公式的集合（如求解三角函数 `sin(x) - 1` 返回的解集）。
>   - **无解**：返回空集 `EmptySet` (显示为 `∅`)。
> - **算法无法解出**：返回条件集 `ConditionSet`（表示数学上知道这是个方程，但无法给出显式解）。

- `solveset` 对同一个根只输出一次，**不包含多重根的重数信息**。如果你需要计算根的多重性，需调用 **`roots()`** 函数，它会返回一个字典 `{根: 重数}`。


## 线性方程组：`linsolve`

用于求解线性系统，支持三种数学上常用的输入形式。解的元组顺序严格对应你传入的符号顺序。

> 1. **表达式列表形式**：`linsolve([x + y + z - 1, x + y + 2*z - 3], (x, y, z))`
> 2. **增广矩阵形式**：传入包含系数和常数项的 `Matrix`。
> 3. **$Ax = b$ 形式**：提取出系数矩阵 $A$ 和常数向量 $b$ 后代入。


## 非线性方程组：`nonlinsolve`
支持求解各类非线性方程组，能够同时处理仅有实数解、仅有复数解、以及拥有无穷多解（返回含参变量的解集）的系统。


> [!CAUTION]
> 目前 `solveset` 和 `nonlinsolve` 仍有功能盲区，遇到以下情况需**回退使用老版本的 `solve()`**：
> 1. **涉及 LambertW 函数的超越方程**（如解 $x e^x - 1 = 0$）。
> 2. **非线性方程组中包含三角函数**。
> 3. `nonlinsolve` 算不出来的带有指数变量的混合非线性组。

# 求解微分方程

SymPy 提供了 **`dsolve`** 用于求解常微分方程 (ODE, 位置函数只含有一个自变量的微分方程)。

- **前置准备：定义未求解的函数**
    在解微分方程前，不能只定义普通变量，必须把 $f(x)$ 声明为一个函数类：
    `f, g = symbols('f g', cls=Function)`
    
- **构建微分方程**
    使用 `f(x).diff(x)` 表示一阶导数 $f'(x)$，`f(x).diff(x, x)` 表示二阶导数 $f''(x)$。通过 `Eq()` 将它们组装起来。
    
- **核心求解：`dsolve`**
    - **语法**：`dsolve(diffeq, f(x))`
    - **返回值**：返回的是一个 `Eq` 等式实例。因为很多复杂的微分方程解无法写出 $f(x) = ...$ 的显式形式，只能给出隐函数表达。
    - **积分常数**：结果中自动生成的任意常数将被统一表示为 `C1`, `C2`, `C3` 等符号。
