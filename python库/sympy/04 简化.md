# 通用化简函数：`simplify()`

- **功能**：智能尝试应用多种底层化简操作，并使用启发式算法确定表达式的“最简”形式。

> **示例**：
    - `simplify(sin(x)**2 + cos(x)**2)` → `1`
    - `simplify((x**3 + x**2 - x - 1)/(x**2 + 2*x + 1))` → `x - 1`
    - `simplify(gamma(x)/gamma(x - 2))` → `(x - 2)*(x - 1)`

> [!CAUTION]	
> **局限与陷阱**：
> 1. **“最简”定义不明确**：例如希望将 $x^2 + 2x + 1$ 变为 $(x + 1)^2$，调用 `simplify` 仍会返回展开的形式（此时应使用 `factor()`）。
>  2. **执行速度较慢**：在选择最佳结果前，它会尝试非常多种类的化简运算。
> 3. **无输出形式保证**：它不保证结果具有特定的数学结构（如不可约因式等），甚至可能错过某些 SymPy 本能做到的特定化简。

- **使用建议**：最适合在交互式环境中探索未知表达式时使用。若明确知道需要何种化简，应直接调用后续的特定函数。

# 多项式与有理函数化简

## 展开：`expand()`

- **功能**：将多项式展开为单项式求和的规范形式

## 因式分解：`factor()` 与 `factor_list()`

- **功能**：在有理数域内，将多项式分解为“不可约”的因式。它是 `expand()` 的逆操作。
- **保证**：调用它处理带有理系数的多项式时，能保证得出的每个因式都是不可约的。
- **结构化输出**：使用 `factor_list()` 可以返回一个包含常数项及 `(因式, 幂次)` 列表的结构化元组。
```python
factor_list(x**2*z + 4*x*y*z + 4*y**2*z)
# (1, [(z, 1), (x + 2⋅y, 2)])
```

## 提取公因式：`collect()`

- **功能**：在表达式中，收集某个指定变量的共同次幂。
- **配套方法**：结合 `.coeff(x, n)` 方法，可以直接提取出表达式中 $x^n$ 对应的系数。

## 约分：`cancel()`
- **功能**：将任意有理函数转化为标准形式 $p/q$。其中分子 $p$ 和分母 $q$ 是没有公因式的展开多项式，且首项系数没有分母（即系数为整数）。
- **性能**：如果目的仅仅是确保表达式被彻底约分，`cancel()` 的运行效率高于 `factor()`。

### 2.5 部分分式分解：`apart()`

- **功能**：对有理函数进行部分分式分解，拆解为多个分式的和。

## 3. 三角函数化简

_(注：SymPy 中的反三角函数按 Python 约定命名，即在前面加 `a`，如 `acos`, `asin`)_

### 3.1 恒等式化简：`trigsimp()`

- **功能**：使用三角恒等式化简表达式，同样使用启发式算法返回“最佳”结果。
    
- **范围**：支持常规三角函数，也支持双曲三角函数（如 `cosh`, `sinh`, `tanh`）。
    

### 3.2 恒等式展开：`expand_trig()`

- **功能**：应用和差角公式或倍角公式展开三角函数（如将 `sin(x + y)` 展开为 `sin(x)*cos(y) + sin(y)*cos(x)`）。它与 `trigsimp()` 的操作方向通常相反。

## 4. 幂运算化简 (Powers)

文档指出了三种幂运算恒等式及其在复数系统中的有效性前提：

1. **$x^a x^b = x^{a+b}$**：始终成立。
    
2. **$x^a y^a = (xy)^a$**：**非**始终成立（反例：$x=y=-1, a=1/2$ 时计算结果冲突）。成立的充分条件是：$x, y \ge 0$ 且 $a$ 为实数。
    
3. **$(x^a)^b = x^{ab}$**：**非**始终成立。成立的充分条件是：$b$ 是整数。
    

基于上述法则，对应以下化简函数：

- **`powsimp()`**：从左至右应用恒等式 1 和 2 合并幂次。
    
- **`expand_power_exp()`**：从右至左应用恒等式 1，拆分指数。
    
- **`expand_power_base()`**：从右至左应用恒等式 2，拆分底数。
    
- **`powdenest()`**：从左至右应用恒等式 3，化简嵌套的幂。
    
- **`force=True` 参数**：如果不希望受限于上述成立条件，可以在调用这些函数时传入 `force=True`，强制忽略变量的安全假设进行化简运算。

## 5. 指数与对数化简

同理，对数恒等式 $\log(xy) = \log(x) + \log(y)$ 与 $\log(x^n) = n\log(x)$ 并非对任意复数都成立，其充分条件是 **$x, y$ 均为正数且 $n$ 为实数**。

- **`expand_log()`**：从左至右应用恒等式，展开对数。
    
- **`logcombine()`**：从右至左应用恒等式，合并对数。
    
- _(这两个函数同样支持使用 `force=True` 强制化简。)_

## 6. 特殊函数化简

SymPy 包含多种特殊函数（如阶乘 `factorial`、组合数 `binomial`、伽马函数 `gamma`、超几何函数 `hyper` 等）。

- **`rewrite()`**：极为通用的方法，将表达式改写为另一种函数形式。例如 `tan(x).rewrite(cos)`，或将阶乘改写为伽马函数 `factorial(x).rewrite(gamma)`。
    
- **`expand_func()`**：根据相关等式将特殊函数展开（如 `gamma(x + 3)` 展开为多项式与 `gamma(x)` 的乘积）。
    
- **`hyperexpand()`**：专门用于将超几何函数 `hyper` 或 Meijer G-函数 重写为更标准的初等函数。
    
- **`combsimp()`**：化简组合数学表达式（处理包含阶乘、组合数的等式）。
    
- **`gammasimp()`**：化简包含伽马函数或带有非整数参数的组合函数的表达式。

## 7. 综合示例：连分数 (Continued Fractions)

文档最后展示了如何利用化简函数处理连分数：

1. 可以通过对列表 `[a0, a1, ..., an]` 进行倒序迭代，结合加法与取倒数操作构建符号化的连分数表达式。
    
2. 使用 **`cancel()`** 可以将嵌套的连分数直接化简为标准的规范分数形式。
    
3. 通过循环调用 **`apart(frac, a_n)`** （提取特定变量的部分分式），可以将规范分数再一层层剥离，还原回原始的连分数结构。
