# 全局配置 (`init_printing`)

`init_printing()` 不仅仅是一个开关，它包含丰富的配置项，用于定义全局的渲染行为。
- **工作机制**：它会自动检测当前终端（IPython, Jupyter, 或纯控制台）并挂载最合适的打印机。
> [!TIP]
> **常用核心参数**：
>- `use_unicode=True/False`：强制或禁用 Unicode 字符渲染。
> - `order='lex' | 'rev-lex'`：控制多项式的排列顺序（按字典序排，或逆字典序排）。
> - `use_latex='mathjax' | 'png' | 'svg'`：在 Jupyter 中强制指定 LaTeX 的渲染引擎。

# 基础打印函数 

针对表达式 $x^2 + \frac{1}{2}$，不同的打印器有截然不同的应用场景：
## **标准字符串 `str(expr)`**
- **输出**：`x**2 + 1/2`
- **工程意义**：这是最基础的 1D 纯文本，结果可以直接被 Python 原生的 `eval()` 或 SymPy 的 `sympify()` 解析回表达式。常用于日志记录（Logging）或数据序列化。

##  **美观打印 `pprint(expr)
- **输出**：在终端画出二维的分数和指数。
- **工程意义**：当在没有 GUI 的服务器终端调试推导逻辑时，`pprint` 是唯一能让你直观检查公式对错的工具。默认使用 Unicode，若终端乱码，需显式传入 `pprint(expr, use_unicode=False)` 降级为 ASCII。

## **LaTeX 源码 `latex(expr)`**
- **输出**：`x^{2} + \frac{1}{2}`
- **进阶参数控制**：你可以通过参数精细控制排版。例如控制乘号的样式 `latex(expr, mul_symbol='dot')`（使用 $\cdot$ 而不是空格），或者让矩阵输出为 `bmatrix` 格式 `latex(matrix, mat_str='bmatrix')`。
