# 替换
- 是有个subs(a, b)来实现将表达式中的所有某物实例替换为其它东西
- 用法如`expr.subs(x, 1)`
> [!NOTE]
>  替换通常出于以下原因:
> 1. 代点算出具体位置的值
> 2. 用另外一个子表达式替换子表达式
> 3. 进行一些受控的简化
- 该用法不会改变表达式(sympy表达式是不可变的, 没有改变表达式的函数)

# sympify
- 用来将字符串转换为表达式
```python
str_expr = "x**2 + 3*x - 1/2"
expr = sympify(str_expr)
```

# evalf
- 将数值表达式计算为浮点数
```python
expr = sqrt(8)
expr.evalf()
# 结果为浮点数形式
```
- 可以一任意精度计算浮点表达式, 例如`evalf(15)`将保留15位小数
- 类似`evalf(subs={x: 2.4})`实现对表达式的数值评估
- 使用chop标志设置是否删除输入误差
> [!NOTE]
> 舍入误差: 取近似值从而减少计算代价
