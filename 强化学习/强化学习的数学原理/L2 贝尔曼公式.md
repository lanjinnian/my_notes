## state value定义
- 一些定义的表示方法:![[Pasted image 20260330184607.png]]
- $G _t$ : Discounted return(折扣回报) , 一个随机变量
- state value: $G _t$的期望值(平均值)
- $v _\pi (s) = \mathbb{E}[G _t \mid S _t = s]$
- 是一个conditional expectation(与状态有关,需要具体的状态)
- $\pi$是一个具体的策略
- state value用来衡量策略的价值
