## state value定义
- 一些定义的表示方法:![[Pasted image 20260330184607.png]]
- $G _t$ : Discounted return(折扣回报) , 一个随机变量
- state value: $G _t$的期望值(平均值)
- $v _\pi (s) = \mathbb{E}[G _t \mid S _t = s]$
- 是一个conditional expectation(与状态有关,需要具体的状态)
- $\pi$是一个具体的策略
- state value用来衡量状态的价值, 价值高说明值得agent前往

# action value
- 从一个状态出发, 并且选择了一个动作后得到的average return
- 利用action value选择在策略中要用什么action
- $q_{\pi}(s, a)=\mathbb{E}\left[G_{t} \mid S_t=s, A_{t}=a\right]$
- 依赖于策略, 不同的策略会得到不同的action value
- 第一项中包含某些随机环境中，同一个状态采取同一个动作，得到的奖励可能是不固定的![[Pasted image 20260403211710.png]]
- 如果知道所有状态下的state value, 就可以计算出所有的action value
- 即使策略告诉我们只能做一个动作, 其它动作的action value同样可以计算
> 计算方法:
> 1. 通过state value来计算(依赖模型)
> 2. 不依赖模型, 直接通过数据来计算
- 状态价值是该状态下所有可能动作价值的加权平均
