## 1. 动机示例 (Motivating Examples)

- 状态值函数 (State value) 可以用来评估一个策略 (policy) 的好坏：如果在所有状态 $s \in \mathcal{S}$ 下都有 $v_{\pi_1}(s) \ge v_{\pi_2}(s)$，那么策略 $\pi_1$ 就比 $\pi_2$ 更好 。
    
- **如何改进策略：** 即使当前策略不够好，我们也可以通过“动作价值 (action values)”来改进它 。
    
- **贪婪策略 (Greedy Policy)：** 计算出某个状态下所有动作的价值 $q_\pi(s, a)$ 后，选择价值最大的动作，即可得到一个改进后的新策略：$\pi_{new}(a|s) = 1$（当 $a = a^*$），其余项为 $0$，其中 $a^* = \arg \max_a q_\pi(s, a)$ 。

> [!NOTE]
> arg max 表示最大值的自变量
    
- 这种做法之所以能提升策略，直观上是因为动作价值评估了动作的好坏；在数学上，则由贝尔曼最优方程 (BOE) 提供理论支撑 。
    

## 2. 最优策略的定义 (Definition of Optimal Policy)

- **定义：** 如果一个策略 $\pi^*$ 满足对于所有状态 $s$ 和任意其他策略 $\pi$，都有 $v_{\pi^*}(s) \ge v_\pi(s)$，那么 $\pi^*$ 就是最优策略 。
    
- 研究贝尔曼最优方程 (BOE) 的目的，就是为了回答以下问题：最优策略是否存在？是否唯一？是随机的还是确定性的？以及如何获取最优策略？
    

## 3. 贝尔曼最优方程简介 (BOE: Introduction)

BOE 描述了最优状态值和最优策略，其形式既优雅又巧妙 。巧妙之处在于等号右侧存在一个最大化 (maximization) 操作，这使得计算变得不那么直观 。

- **元素形式 (Elementwise form):**
    $$v(s) = \max_\pi \sum_a \pi(a|s) \left( \sum_r p(r|s,a)r + \gamma \sum_{s'} p(s'|s,a)v(s') \right)$$
    对所有 $s \in \mathcal{S}$ 成立，也可以简写为 $v(s) = \max_\pi \sum_a \pi(a|s) q(s,a)$ 。

- **矩阵-向量形式 (Matrix-vector form):**
    
    $$v = \max_\pi (r_\pi + \gamma P_\pi v)$$
    
    其中，向量的元素定义为 $[r_\pi]_s \triangleq \sum_a \pi(a|s) \sum_r p(r|s,a)r$，概率转移矩阵的元素定义为 $[P_\pi]_{s,s'} \triangleq \sum_a \pi(a|s) p(s'|s,a)$ 。
    

## 4. 处理方程右侧的最大化 (Maximization on the Right-hand Side)

- 为了求解 $\max_\pi \sum_a \pi(a|s) q(s,a)$，我们需要考虑概率之和为 1 的约束，即 $\sum_a \pi(a|s) = 1$ 。
    
- 这相当于在给定的 $q(s,a)$ 中寻找最大值：$\max_\pi \sum_a \pi(a|s) q(s,a) = \max_{a \in \mathcal{A}(s)} q(s,a)$ 。
    
- 当策略变为确定性贪婪策略时达到最优，即当 $a = a^*$ 时 $\pi(a|s) = 1$（其余为 0），其中 $a^* = \arg \max_a q(s,a)$ 。
    

## 5 & 6. 压缩映射定理与 BOE 的重写 (Contraction Mapping Theorem & Rewrite)

为了证明 BOE 的解的存在性和唯一性，引入了压缩映射定理。

- **重写 BOE：** 令 $f(v) := \max_\pi (r_\pi + \gamma P_\pi v)$，则 BOE 可以简写为求不动点的问题：$v = f(v)$ 。
    
- **压缩映射概念：** 如果函数 $f$ 满足 $\|f(x_1) - f(x_2)\| [cite_start]\le \gamma \|x_1 - x_2\|$，且 $\gamma \in (0,1)$，则 $f$ 是一个压缩映射 (Contraction mapping) 。满足 $f(x) = x$ 的点称为不动点 (Fixed point) 。
    
- **压缩映射定理：** 对于 $x = f(x)$，如果 $f$ 是压缩映射，那么：
    
    - **存在性：** 存在一个不动点 $x^*$ 。
        
    - **唯一性：** 这个不动点是唯一的 。
        
    - **算法：** 构造序列 $x_{k+1} = f(x_k)$，当 $k \to \infty$ 时，序列以指数速度收敛于 $x^*$ 。
        
- **BOE 的收缩性 (Contraction Property)：** 定理表明，$f(v)$ 是一个压缩映射，满足 $\|f(v_1) - f(v_2)\| [cite_start]\le \gamma \|v_1 - v_2\|$，这里的 $\gamma$ 正好就是强化学习中的折扣率 (discount rate)！ 。
    

## 7. 求解贝尔曼最优方程 (Solving the BOE)

根据压缩映射定理，BOE 必然存在唯一的解 $v^*$，且可以通过迭代算法求解 。这个迭代算法实际上就是下一章会讲到的**值迭代算法 (Value Iteration Algorithm)** 。

- **迭代公式：** $v_{k+1} = f(v_k) = \max_\pi (r_\pi + \gamma P_\pi v_k)$ 。
    
- **具体步骤 (Procedure summary)：**
    
    1. 针对任意状态 $s$，获取当前的估计值 $v_k(s)$ 。
        
    2. 计算所有动作的 $q$ 值：$q_k(s,a) = \sum_r p(r|s,a)r + \gamma \sum_{s'} p(s'|s,a)v_k(s')$ 。
        
    3. 计算贪婪策略：$\pi_{k+1}(a|s) = 1$ (当 $a = \arg \max_a q_k(s,a)$ 时) 。
        
    4. 更新状态值：$v_{k+1}(s) = \max_a q_k(s,a)$ 。
        

## 8. BOE 与最优性 (BOE: Optimality)

我们之所以研究 BOE，是因为它的解直接对应着最优状态值和最优策略 。

- **策略最优性定理 (Policy Optimality)：** 假设 $v^*$ 是 BOE 的唯一解，那么对于任意给定策略 $\pi$ 的状态值 $v_\pi$，都有 $v^* \ge v_\pi$ 。
    
- **贪婪最优策略 (Greedy Optimal Policy)：** 对于任意 $s \in \mathcal{S}$，确定性的贪婪策略 $\pi^*(a|s) = 1$ (当 $a = \arg \max_a q^*(s,a)$) 就是求解 BOE 的一个最优策略 。
    

## 9. 分析最优策略 (Analyzing Optimal Policies)

从 BOE 公式中可以看出，决定最优策略的因素有三个：**奖励设计 ($r$)**、**系统模型 ($p(s'|s,a), p(r|s,a)$)** 和 **折扣率 ($\gamma$)** 。

- **折扣率 ($\gamma$) 的影响：**
    
    - 较高的 $\gamma$ (如 0.9)：策略具有远见，甚至敢于冒险进入惩罚区域以获取最终的高奖励 。折扣率使得策略不会去走无意义的弯路 (meaningless detour) 。
        
    - 较低的 $\gamma$ (如 0.5)：策略变得短视 (shorted-sighted)，会避开所有惩罚区域 。
        
    - 极端的 $\gamma = 0$：策略极其短视，只追求最大化即时奖励，可能永远无法到达目标 。
        
- **奖励 ($r$) 的影响：** 增加对危险区域的惩罚（例如将 $r_{forbidden}$ 从 -1 调至 -10），最优策略会主动避开这些区域 。
    
- **最优策略不变性定理 (Optimal Policy Invariance)：**
    
    - 起决定作用的不是绝对的奖励数值，而是它们的**相对值** 。
        
    - 如果对所有的奖励 $r$ 进行仿射变换 (affine transformation)，变为 $ar + b$（其中 $a \neq 0$），那么对应的最优状态值 $v'$ 也会发生仿射变换：$v' = av^* + \frac{b}{1-\gamma}\mathbf{1}$ 。
        
    - **结论：** 最优策略对奖励信号的仿射变换具有不变性 (invariant) 。
