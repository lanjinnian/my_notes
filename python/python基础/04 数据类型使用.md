### 1. Python3 数字 (Number) 高阶操作

除了基本的加减乘除，Python 提供了强大的数学和随机数处理模块。在使用高级数学函数前，通常需要导入 `math` 或 `random` 模块。

**常用数学函数 (`import math`)：**
* `abs(x)`: 返回数字的绝对值（内置函数，不需要导包）。
* `round(x, n)`: 返回浮点数 x 的四舍五入值，保留 n 位小数（内置函数）。
* `math.ceil(x)`: 向上取整，如 `math.ceil(4.1)` 返回 `5`。
* `math.floor(x)`: 向下取整，如 `math.floor(4.9)` 返回 `4`。

**常用随机数函数 (`import random`)：**
* `random.random()`: 随机生成下一个实数，范围在 `[0, 1)` 之间。
* `random.randint(a, b)`: 生成一个指定范围 `[a, b]` 内的整数。
* `random.choice(seq)`: 从序列（列表、元组、字符串）中随机挑选一个元素。
* `random.shuffle(lst)`: 将列表的所有元素随机打乱。

### 2. Python3 字符串 (String) 核心方法

字符串是不可变的，因此所有的字符串修改方法**都会返回一个新字符串**，而不会改变原字符串。

**字符串格式化 (强烈推荐 f-string)：**
Python 3.6+ 引入了极简的 f-string 格式化方法，在字符串前加 `f`，即可直接在大括号 `{}` 中放入变量名或表达式。
```python
name = "Alice"
age = 20
print(f"我的名字是 {name}，明年 {age + 1} 岁。") 
```

**极其高频的内置方法：**
* `str.split(sep)`: 将字符串以 `sep` 为分隔符拆分成**列表**。
* `str.join(seq)`: 将序列（如列表）中的元素以指定的字符连接生成一个新的**字符串**。
* `str.strip(chars)`: 去除字符串头尾指定的字符（默认为空格或换行符）。
* `str.replace(old, new)`: 将字符串中的 `old` 替换成 `new`。
* `str.upper()` / `str.lower()`: 全部转换为大写 / 小写。
* `str.find(sub)`: 查找子串，找到返回起始索引，找不到返回 `-1`。

```python
# join 和 split 是黄金搭档
words = ["I", "love", "Python"]
sentence = " ".join(words)  # 输出: "I love Python"
print(sentence.split(" "))  # 输出: ['I', 'love', 'Python']
```

### 3. Python3 列表 (List) 核心方法

列表是可变的，以下方法通常会**直接在原列表上进行修改**（就地修改），不会返回新列表（返回 `None`）。

**增删改查：**
* **增**:
    * `list.append(obj)`: 在列表末尾添加新的**一个**对象。
    * `list.extend(seq)`: 在列表末尾一次性追加另一个序列中的**多个**值。
    * `list.insert(index, obj)`: 将对象插入到指定索引位置。
* **删**:
    * `list.pop([index])`: 移除列表中的一个元素（默认最后一个元素），并且**返回该元素的值**。
    * `list.remove(obj)`: 移除列表中某个值的**第一个**匹配项。
    * `del list[index]`: 使用 `del` 语句通过索引删除。
* **排序与翻转**:
    * `list.sort(reverse=False)`: 对原列表进行升序排序（设为 True 为降序）。
    * `list.reverse()`: 反向列表中元素。

### 4. Python3 元组 (Tuple) 进阶特性

元组的核心特性就是**不可变**，因此它没有 `append`、`remove` 等修改方法。它的常用操作通常仅限于：
* `len(tup)`: 计算元组元素个数。
* `max(tup)` / `min(tup)`: 返回元组中元素最大/最小值。

**⚠️ 经典陷阱：元组中的可变对象**
虽然元组本身不可变，但如果元组的某个元素是**可变对象**（如列表），那么该可变对象内部的内容是可以被修改的！
```python
tup = (1, 2, [3, 4])
# tup[0] = 100  # 这会报错，因为元组的元素指向不能变
tup[2].append(5) # 这是合法的！改变的是列表内部
print(tup)       # 输出: (1, 2, [3, 4, 5])
```

### 5. Python3 集合 (Set) 核心方法

集合（Set）的主要应用场景是**去重**和**关系测试**（数学里的交、并、差集）。

**修改集合：**
* `set.add(elm)`: 为集合添加**一个**元素（已存在则不进行任何操作）。
* `set.update(set2)`: 添加**多个**元素（可以是列表、元组、字典等）。
* `set.remove(elm)`: 移除元素，如果元素不存在会**报错 (KeyError)**。
* `set.discard(elm)`: 移除元素，如果元素不存在**不会报错**（比 remove 更安全）。

**数学集合运算：**
```python
set_A = {1, 2, 3, 4}
set_B = {3, 4, 5, 6}

print(set_A - set_B) # 差集: {1, 2} (在A中但不在B中)
print(set_A | set_B) # 并集: {1, 2, 3, 4, 5, 6} (包含A和B的所有元素)
print(set_A & set_B) # 交集: {3, 4} (同时在A和B中的元素)
print(set_A ^ set_B) # 对称差集: {1, 2, 5, 6} (不同时在A和B中的元素)
```
