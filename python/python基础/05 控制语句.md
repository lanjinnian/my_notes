## 1. 条件控制 (if...elif...else)

Python 依靠条件语句的执行结果（`True` 或 `False`）来决定执行哪个代码块。与其他语言不同，Python 完全依赖**缩进**来划分代码块，而不是大括号。

### 基础语法

Python

```
if condition_1:
    # 如果 condition_1 为 True，执行这里的代码
    statement_block_1
elif condition_2:
    # 如果 condition_1 为 False 且 condition_2 为 True，执行这里
    statement_block_2
else:
    # 如果以上所有条件都为 False，执行这里
    statement_block_3
```

> **注意**：
> 
> 1. 每个条件后面必须有冒号 `:`。
>     
> 2. `elif` 是 `else if` 的缩写，可以有零个或多个 `elif`。
>     
> 3. `else` 是可选的，最多只能有一个。
>     

### 代码示例

Python

```
age = int(input("请输入你家狗狗的年龄: "))

if age <= 0:
    print("你是在逗我吧！")
elif age == 1:
    print("相当于 14 岁的人类。")
elif age == 2:
    print("相当于 22 岁的人类。")
else:
    human_age = 22 + (age - 2) * 5
    print(f"对应人类年龄: {human_age} 岁。")
```

### 💡 扩展：`match...case` (Python 3.10+ 新特性)

如果你使用的是较新的 Python 版本，可以使用类似于其他语言中 `switch...case` 的语法进行模式匹配：

Python

```
status = 404
match status:
    case 400:
        print("Bad request")
    case 404:
        print("Not found")
    case _:  # 下划线表示通配符，相当于 default 或 else
        print("Something's wrong")
```

## 2. 循环语句 (while & for)

Python 提供了两种主要的循环结构：`while` 循环和 `for` 循环。

### ① `while` 循环

只要条件为 `True`，`while` 循环就会一直执行内部的代码块。通常用于**事先不知道要循环多少次**的场景。

Python

```
n = 1
sum_val = 0

# 计算 1 到 100 的和
while n <= 100:
    sum_val += n
    n += 1

print(f"1 到 100 的和为: {sum_val}")
```

> **死循环警告**：如果 `while` 的条件永远不变为 `False`（例如忘记更新计数器变量），程序将无限运行下去。可以通过 `Ctrl + C` 来强制中断。

### ② `for` 循环

Python 的 `for` 循环非常强大，它主要用于**遍历任何序列项目**（如列表、字符串、元组或字典）。

Python

```
languages = ["C", "C++", "Perl", "Python"] 
for x in languages:
    print(x)
```

### ③ `range()` 函数

如果你需要遍历一个数字序列，内置的 `range()` 函数非常方便。它可以生成数列。

- `range(5)`: 生成 `0, 1, 2, 3, 4`（不包含结束值 5）。
    
- `range(2, 6)`: 生成 `2, 3, 4, 5`。
    
- `range(0, 10, 3)`: 生成 `0, 3, 6, 9`（指定步长为 3）。
    

Python

```
for i in range(3):
    print(f"这是第 {i} 次循环")
```

## 3. 循环控制语句 (break, continue, pass)

在循环执行的过程中，有时我们需要强行改变循环的默认流程。

- **`break`**: **打破整个循环**。立即退出当前的 `for` 或 `while` 循环，不再执行循环体中剩余的代码。
    
- **`continue`**: **跳过本次循环**。停止执行当前这一次循环中剩余的代码，直接进入下一次循环条件的判断。
    
- **`pass`**: **占位符**。它什么也不做，通常用于保证语法的完整性（当你还没想好这里写什么代码，但不写又会报错时使用）。
    

### `break` 与 `continue` 对比示例

Python

```
print("--- 测试 break ---")
for letter in 'Python':
    if letter == 'h':
        break  # 遇到 'h' 直接结束整个循环
    print(f"当前字母: {letter}")
# 输出: P, y, t

print("\n--- 测试 continue ---")
for letter in 'Python':
    if letter == 'h':
        continue # 遇到 'h' 跳过这一次，继续下一个字母
    print(f"当前字母: {letter}")
# 输出: P, y, t, o, n
```

## 4. 独门秘籍：循环中的 `else` 语句

这是 Python 语言中一个相对独特且很多人容易误解的语法：`for...else` 和 `while...else`。

- **触发机制**：只有当循环**正常执行完毕**（即没有被 `break` 强行中断）时，`else` 块里面的代码才会被执行。
    
- **应用场景**：常用于搜索操作。如果在循环里找到了目标（触发 `break`），就不执行 `else`；如果整个循环跑完了都没找到，就执行 `else`。
    

Python

```
# 寻找质数的经典示例
for num in range(10, 20):  # 迭代 10 到 19
    for i in range(2, num): # 根据因子迭代
        if num % i == 0:    # 确定第一个因子
            j = num / i     # 计算第二个因子
            print(f"{num} 等于 {i} * {j}")
            break # 找到了因子，不是质数，打断内层循环
    else: 
        # 内层循环正常跑完（没有触发 break），说明没找到因子
        print(f"{num} 是一个质数")
```
